

# node.js 기반 typescript 풀스택 프레임워크 개인 프로젝트
    - 프레임워크 : node.js, exrpess
    - 개발 언어 : typescript, ajax, json, jqeury
    - 기능
        
        1) 로그인
        
        - ID와 PW 확인
        - 휴면 계정 확인
        - 회사의 유효 기간 확인
        - 로그인 시간 업데이트
        
        2) 회원 가입
        
        - ID 중복 검사
        - 정규식을 이용한 PW의 유효성 검사
        - 회사 코드 선택 및 관리자와 사용자 선택
        - Read & Write 권한 선택
        
        3) 회원 목록 조회
        
        - Read 권한과 관리자 계정만 조회 가능
